# asdf-protoc-gen-go-temporal


[protoc-gen-go-temporal](https://github.com/cludden/protoc-gen-go-temporal) as an [asdf](https://asdf-vm.com) plugin.

## Install

```bash
asdf plugin-add protoc-gen-go-temporal https://github.com/yourusername/asdf-protoc-gen-go-temporal.git
```

## USE

asdf list-all protoc-gen-go-temporal
asdf install protoc-gen-go-temporal <version>
asdf global protoc-gen-go-temporal <version>
protoc-gen-go-temporal --version
